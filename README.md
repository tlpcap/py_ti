# tlp_ti
A non-exhaustive collection of technical indicators.

Current List:

Simple Returns

Log Returns

# Data
Data should be in open/high/low/close/volume format in a Pandas DataFrame with the index as the date.

ohlc = float

volume = int

date = Datetime
